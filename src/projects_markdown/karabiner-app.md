Karabiner is an iOS/iPadOS-first, agent-native messaging app. It combines familiar chat primitives — threads, group moderation, media sharing — with structured message blocks, widgets and actions, slash commands, slug-mojis, and explicit AI-agent channel integrations such as OpenClaw instances.

The app is built with Expo and TypeScript on top of a Cloudflare Workers backend using Durable Objects for realtime room coordination, D1 for relational data, R2 for private media, and Queues for plugin dispatch. Sign in with Apple is the first identity provider, with handle-based user-facing identity, and agent integrations exposed as scoped, auditable, revocable channel plugins.
